## 功能列表

### | Neuro Draw（今日运势）

一个简单的抽签插件，每天可以抽取一次今日运势，感受一下 Neuro-sama 式的运气！
<p>发送【抽签】即可获取今日运势哦</p>

### | 碧蓝档案塔罗牌

- ba塔罗牌：随机发送一张ba塔罗牌
- ba占卜：默认以合并转发的方式随机发送一个牌阵进行占卜
- ba运势：随机发送一张ba塔罗牌以及对应的运势分数和对应运势评价
- ba塔罗牌解读：发送一张ba塔罗牌以及塔罗牌原画师的解读

### | 随机表情包

<p>拥有各式各样的表情包，发送对应指令即可，欢迎找【浅浅小饼干】投稿表情包喵，现有的分类如下：</p>
- 猫猫虫
- 龙
- 狗妈
- 柴郡
- 罕见
- 唐氏
- chiikawa

### | maimaiDX

可使用各种功能的maima查分器
<p>可输入【帮助maimaidx】查看具体功能</p>
![image-maimaiDX](img\maimaiDX.png)

### | Today Waifu（今日老婆）

想随机抓一个群友当老婆吗？
<p>发送【今日老婆】可随机抽取一位群友作为您的今日老婆</p>

### | shindan

使用 ShindanMaker 网站的无聊趣味占卜
<p>发送【占卜列表】即可获取可使用功能</p>

### | MD_Bot

这是一款由【Iris】所制作的，专为Muse Dash这款游戏所服务的查分功能
<p>输入【md help】可获取其具体功能</p>
<p>或点击链接[iRis Bot](https://docs.loveiris.cn/#/)查看具体功能</p>

### | Chieri_Bot

chieri bot是一款功能型机器人, 主要提供音游查询服务
<p>而在浅小肆这里主要已【CS点数功能】、【OSU！】、【中二节奏】为主</p>
<p>具体功能可点击链接[Chieri bot docs](https://chieri.docs.chinosk6.cn/channel/)查看具体功能，部分功能无法使用已屏蔽</p>

### | HarukiBot

HarukiBot是一款功能型机器人, 目前主要提供《世界计划 多彩舞台》相关查询服务。

<strong>功能如下</strong>

#### 1. 歌曲信息查询
| 指令 | 参数 | 示例 | 说明 |
|------|------|------|------|
| <p>`pjskinfo <曲名>`</p><p>`song <曲名>`</p>`musicinfo <曲名>` | 支持官方名称或昵称 | <p>`pjskinfo ロキ`</p>`song テオ` | 查询歌曲完整信息（作者、时长等） |
| `pjskbpm <曲名>` | 需准确名称 | `pjskbpm 39みゅーじっく！` | 查询歌曲BPM |
| `查bpm <数字>` | 整数或小数 | `查bpm 180` | 查询匹配BPM的所有歌曲 |

#### 2. 谱面预览
| 指令 | 参数 | 示例 | 说明 |
|------|------|------|------|
| `谱面预览 <曲名> <难度>` | 支持缩写 | `谱面预览 ヴァンパイア ex` | 查询指定难度谱面 |
| `谱面预览 <曲名>` | 默认master | `谱面预览 ドラマツルギー` | 查询master难度谱面 |

**难度缩写表**:
- `easy` → `ez`
- `normal` → `nm`
- `hard` → `hd` 
- `expert` → `ex`
- `master` → `ma`
- `append` → `ap/apd`

#### 3. 昵称设置
| 指令 | 参数 | 示例 | 说明 |
|------|------|------|------|
| `musicset <昵称> to <歌名>` | 自定义昵称 | `musicset 萝卜歌 to だんだん早くなる` | 设置歌曲昵称 |
| `musicdel <昵称>` | 需存在昵称 | `musicdel 萝卜歌` | 删除歌曲昵称 |
| `charaset <昵称> to <角色>` | 角色官方名/昵称 | `charaset knd to 宵崎奏` | 设置全局角色昵称 |
| `charadel <昵称>` | 需存在昵称 | `charadel knd` | 删除全局角色昵称 |
| `grcharaset <新昵称> to <已有昵称>` | 仅当前群 | `grcharaset 葱 to 初音ミク` | 设置群专属角色昵称 |
| `grcharadel <昵称>` | 需群内存在 | `grcharadel 葱` | 删除群专属角色昵称 |

#### 4. 活动排名查询
| 指令 | 参数 | 示例 | 说明 |
|------|------|------|------|
| `wlsk <排名>` | 1~10000 | `wlsk 100` | 查询World Link单榜排名 |
| `wlsk <排名> -c <角色>` | 指定角色 | `wlsk 50 -c haruka` | 查询角色单榜排名 |
| `sk` | 需绑定ID | `sk` | 查询自己前100排名 |
| `sk <游戏ID>` | 玩家UID | `sk 123456789` | 查询指定玩家排名 |
| `sk <排名>` | 1~7个排名 | `sk 1 10 100` | 批量查询排名 |
| `查房 <排名>` | 整数 | `查房 500` | 查询排名最近1小时变动 |
| `sk线` | 无 | `sk线` | 查询当前榜线分数 |
| `时速`/`半日速`/`日速` | 可选分钟数(1-1440) | `时速 30` | 查询PT增长速度 |
| `分数线 <排名>` | 整数 | `分数线 2000` | 查询分数历史趋势 |
| `追踪`/`ptr` | 需绑定ID | `追踪` | 追踪自己PT/排名趋势 |
| `追踪 <排名1> <排名2>` | 双排名 | `追踪 1 2` | 对比两个排名趋势 |

#### 5. 玩家信息查询
| 指令 | 服务器 | 示例 | 说明 |
|------|--------|------|------|
| `绑定 <ID>` | 日服 | `绑定 UID` | 绑定账号 |
| `cn绑定 <ID>` | 国服 | `cn绑定 UID` | 绑定国服账号 |
| `en绑定 <ID>` | 国际服 | `en绑定 UID` | 绑定国际服账号 |
| `tw绑定 <ID>` | 台服 | `tw绑定 UID` | 绑定台服账号 |
| `kr绑定 <ID>` | 韩服 | `kr绑定 UID` | 绑定韩服账号 |
| `pjskprofile` | 日服 | `pjskprofile` | <p>生成个人资料卡片</p>可根据注意事项所写的不同服务器查询其他服数据 |
| `sk` | 日服 | `sk` | <p>查询活动PT（此处为日服）</p>可根据注意事项所写的不同服务器查询其他服数据 |
| `逮捕@[用户]` | 需公开 | `逮捕@[浅浅]` | 查询玩家FC/AP数 |
| `逮捕+<ID>` | 玩家UID | `逮捕 UID` | 查询指定UID信息 |

#### 6. 隐私控制
| 指令 | 说明 |
|------|------|
| `不给看` | 隐藏自己的UID |
| `给看` | 公开自己的UID |

##### 7. 卡牌&活动查询
| 指令 | 参数 | 示例 | 说明 |
|------|------|------|------|
| `查卡面 <角色>` | 角色昵称 | `查卡面 宵崎奏` | 查询角色所有卡牌 |
| `cardinfo <ID>` | 卡牌ID | `cardinfo 1001` | 查询卡牌详情 |
| `event` | 无 | `event` | 查询当前活动 |
| `event <ID>` | 活动ID | `event 52` | 查询指定活动 |
| `findevent <关键词>` | 多关键词 | `findevent 紫月 knd` | 筛选活动 |
| `活动图鉴` | 无 | `活动图鉴` | 查询全活动概览 |

**活动查询关键词**:
- 类型：`5v5`、`马拉松`、`普活`
- 属性：`紫月`、`蓝星`、`草`（绿）、`赤热`
- 角色：`knd`（宵崎奏）、`miku`、`25h`（25时成员）、`25miku`（白葱）
- 组合：`knd ick`（同时含宵崎奏和晓山瑞希）

#### 注意事项
1. 所有昵称设置会被记录，违规可能导致功能禁用
2. 谱面预览数据来自第三方
3. 服务器前缀：
   - `en`：国际服
   - `tw`：台服 
   - `kr`：韩服
   - `cn`：国服
4. 活动图鉴因图片过大无法在频道使用

### | Arcaea 表情包生成器

可以生成 Arcaea 风格的表情包，支持自定义文字、位置、角度、颜色等参数
- arc [角色] [文字] —— 生成表情包
- arc -h —— 显示帮助
- arc —— 进入交互模式

| 参数 | 说明 | 范围 | 默认值 | 补充说明 |
| :-----: | :----: | :----: | :----: | :----: |
| -s, --size | 文字大小 | 20~45 | 35 | 数字越大文字越大,多行文字建议25-35 |
| -x | 横向位置 | 0~296 | 148 | 0=最左边,148=居中,296=最右边 |
| -y | 纵向位置 | 0~256 | 128 | 0=最上方,128=居中,256=最下方 |
| -r, --rotate | 旋转角度 | -180~180 | -12 | 正数顺时针,负数逆时针,建议-30~30度 |
| -c, --color | 文字颜色 | 十六进制	 | 角色专属 | 支持#ff0000或ff0000格式 |
| -w, --stroke-width | 描边宽度	 | 整数 | 9 | 文字边框的粗细 |
| -C, --stroke-color | 描边颜色	 | 	十六进制 | 自动生成 | 默认比文字颜色深30% |

<strong>💡 提示: 文字包含空格需要加引号,换行使用\n</strong>
